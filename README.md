# Movie Recommender System
A content-based movie recommendation system that suggests similar movies based on their metadata, such as genres, keywords, cast, and crew. 


### Dataset link
[The Movies Dataset](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset/data)

### Features
- Feature extraction using tokenization, stemming, and stop-word removal.
-  TF-IDF vectorization to convert text data into numerical format.
-  Cosine similarity to compute movie similarity scores.
